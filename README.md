# Portafolio - Marlon Castillo

Este es el repositorio del portafolio personal de Marlon Castillo, Desarrollador Web. 
Construido con buenas prácticas, el portafolio muestra proyectos, habilidades técnicas y una galería especial de mascotas.

## Tecnologías Utilizadas

- **HTML5** para la estructura y semántica.
- **CSS3** para los estilos, layout (CSS Grid y Flexbox) y diseño responsivo, incluyendo soporte de variables y modo claro/oscuro.
- **JavaScript** para la interactividad de la página (menú de navegación, cambio de tema, etc.).

## Estructura del Proyecto

- `index.html`: Página principal con presentación, sobre mí, proyectos y formulario de contacto.
- `mascotas.html`: Galería de mascotas organizada en una cuadrícula (grid) adaptativa.
- `css/style.css`: Estilos globales y específicos del sitio web.
- `js/script.js`: Archivo con la lógica interactiva del lado del cliente.
- `images/`: Directorio con los recursos gráficos (imágenes webp, svg, etc.).

## Visualización Local

Para ver el proyecto en tu máquina local, simplemente puedes abrir el archivo `index.html` en tu navegador web de preferencia, o utilizar herramientas como Live Server en Visual Studio Code.
